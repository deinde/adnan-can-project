# Directives

This folder contains SOPs (Standard Operating Procedures) written in Markdown.

Each directive defines:
- **Goal**: What needs to be accomplished
- **Inputs**: What data or resources are needed
- **Tools/Scripts**: Which execution scripts to use (from `execution/`)
- **Outputs**: What the expected deliverable is
- **Edge Cases**: Known gotchas, API limits, timing issues

Directives are living documents — they get updated as the system learns.
