# Directive 001: UI Development - Core Layout

## Goal
Build the initial frontend structure for the Concentrates Marketplace.

## Inputs
- `directives/project_blueprint.md`

## Steps
1. Initialize a web application (Vite + React suggested).
2. Implement the **Top Navigation Bar**.
3. Implement the **Left Sidebar** with category navigation and filters.
4. Build the **Main Marketplace View** skeleton with a toggle for Grid/Table views.
5. Apply the "Industrial, Data-driven" CSS design system.

## Tools/Scripts
- `npx create-vite@latest ./ --template react`
- `npm install`
- Manual CSS/JS implementation in `src/`

## Outputs
- A running local dev server with the core layout visible.
- Responsive design for Browse, Suppliers, and Orders.
