# Directive 004: Market Analytics & Reports Page

## Goal
Implement a standalone analytics page for procurement tracking and market pricing trends.

## Inputs
- `directives/project_blueprint.md`

## Steps
1. Create a "Reports/Analytics" view in `App.jsx`.
2. Implement **Pricing Trend Charts** (visualized using industrial data tables or simple SVG graphs).
3. Add **Procurement History** (Previous orders, total spend).
4. Integrate with the "Analytics" link in the Navbar.

## Tools/Scripts
- Vite/React.

## Outputs
- Data-driven analytics page that matches the platform's high-tech aesthetic.
