# Directive 002: Seller Dashboard Implementation

## Goal
Build the Seller Interface for managing inventory and viewing analytics.

## Inputs
- `directives/project_blueprint.md`

## Steps
1. Create a `src/components/SellerDashboard.jsx` component.
2. Implement the **Inventory Overview** (Active Batches).
3. Implement the **Listing Analytics** (Views, Offer Rate).
4. Add a "Toggle Mode" in the Navbar to switch between Buyer and Seller views.

## Tools/Scripts
- Vite/React implementation.

## Outputs
- Functional Seller Dashboard view with industrial charts/data.
